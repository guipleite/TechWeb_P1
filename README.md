# TechWeb_P1 - EventNote 


# Proposta: 

-Para esse projeto, foi proposto a criação de CRUDS que interagissem com uma arquitetura MVC e uma base de dados no MySQL, 
se inspirando em aplicações de blocos de notas conhecidas no mercado.

# Sobre o EventNote:

-O EventNote possui dois CRUDS. O primeiro trata de um banco de dados para usuários da aplicação. O segundo trata da 
aplicação em sí, um bloco de nota para eventos, que funciona como uma agenda. Nele o usuário que possui sua conta pode
adicionar eventos a sua agenda(presente no seu display) que estará organizada em ordem cronológica e por categorias 
relacionadas a necessidade da tarefa. Ainda é possível usufruir de telas de edição de eventos, busca por título e/ou 
conteúdo, ou até mesmo filtro por datas. Por fim o usuário ainda pode manipular seus dados da conta e inclusive 
delata-la, perdendo assim todos seus eventos antes criados.
